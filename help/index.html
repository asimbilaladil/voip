Installation

Automatic installation

To add the Order Now Form using the built-in plugin installer:

  • Step 1: Upload the plugin files in your server
  • Step 2: Copy the link in your browser eg: www.yourwebsite/php-orderform
  • Step 3: You can see a screen like below

Step 4: Agree the licence by clicking the check box and click continue button.

Step 5:  You will get a screen like shown below

It shows your php version ,Session, MYSQL and config file status

Step 6: Click 'continue' button it shows the screen like below

 Here you have to fill out

Site Name - Your website name

Database Name - Database name you created in your  server.

Table Prefix - Its optional

Database Host - By default it will be localhost

Database Username - Username you used for creating database

Database Password - Password you used for creating database

Admin username - Admin name you needed

Admin password - Admin password you needed (you can change it later)

Email id - Your Email id

After filling these details, you will redirect to a page with admin page link and front page link.

Some sample data is installed with the installation.

In admin page, you can login using the username and password you provided.      

Upgrade Process

Upgrade Steps

Make a full backup of all files, directories, and your database before starting, and save it outside your Order Form installation directory.

Remove all old files and directories, except the followings:

- common/config/config.php: Stores the database configuration.

- public/uploads: Stores files uploaded.

- Any custom directories and files you added elsewhere.

- If you made changes in index.php or css/style.css , copy that changes from backup file and apply in new file.

If you made modifications to files like .htaccess or robots.txt or config/config.php, you will need to re-apply them from your backup, after the new files are in place. upload the latest Order Form release to your Order Forms directory, except for the following files and directories:

common/config/config.php

public/uploads

And that’s it! The upgrade is now complete. Enjoy!

Note: It is always recommended try an upgrade on a test copy of your site before applying it to your live site. Even minor updates can cause your site's behavior change.

Order Form Settings

Order to Email - This is the email id you want to get order mails.

Paypal Link - There is a dropdown with two links one is paypal url and other is paypal sandbox test url.

For testing purpose you can use sandbox test url.

Paypal business Email - This is the paypal business email (Your Paypal id)

Here you can configure SMTP mails by checking Use SMTP

You can fill out SMTP Host ,SMTP Username, SMTP Password, SMTP Port

Bottom Content - This is the content you want to show at front end as terms and conditions , which shows above 'Order Now' button.
